# KittenCV - Computer Vision для эмоций с котиками  **KittenCV** - веб-приложение компьютерного зрения, которое определяет  выражения лица и показывает соответствующие реакции котиков в реальном времени.  ## Технологии: - **Computer Vision** (Mediapipe, OpenCV) - **HTTPS** веб-сервер - **Real-time** обработка с веб-камеры
